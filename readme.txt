##HOW TO USE:
1)Compile the three java file(javac gui.java ftpserver.java ftpclient.java)
2)run the ftpserver class file first (java ftpserver)
3)run the ftpclient class file (java ftpclient)
4)select the file from the gui interface (sorry for the design and color need to be fixed.It only support the txt file currently wil be extended to other file formate in future)
5)the text file will be transfered to the pc which was running the server script.
